# FlowDoc FAQ

## 1. 参赛与提交

| 问题 | 回答 |
|---|---|
| 参赛形式有要求吗？ | 可以个人或 3 人以内组队参赛。 |
| 报名时就要提交作品吗？ | 是，报名阶段就需要准备并提交作品。 |
| 提交后还能修改吗？ | 不能，提交后作品不可修改。 |
| 截止时间是什么时候？ | 以官方公布的截止时间为准，初赛截止到北京时间 2026 年 5 月 18 日 23:59。 |
| 提交材料怎么命名？ | 按赛道名、选手名、作品名等规范命名。 |
| 演示录屏必须有语音吗？ | 必须有，建议配音或字幕说明。 |

## 2. 作品形态

| 问题 | 回答 |
|---|---|
| 一定要是完整上线产品吗？ | 不需要，Demo 原型也可以，但要能看出产品闭环。 |
| 可以用第三方 AI 平台或工具吗？ | 可以，评审更看重产品创意和 AI 能力的应用方式。 |
| 可以用 Vercel / Netlify / GitHub Pages 部署吗？ | 可以，只要能提供可访问的 Demo 链接。 |
| Demo 链接如果需要登录怎么办？ | 需要在说明文档里提供体验账号和密码。 |
| 可以交 Figma / Axure / 纯原型吗？ | 可以，但建议尽量做到可交互、可演示。 |

## 3. 赛道与评审

| 问题 | 回答 |
|---|---|
| 非计算机专业可以参赛吗？ | 可以，比赛看重产品创意和理解能力。 |
| 不会写代码可以参赛吗？ | 可以，初赛更看重 Demo、演示录屏和说明文档。 |
| 可以同时参加命题赛道和开放赛道吗？ | 不可以，只能选择一个。 |
| 评审主要看什么？ | 主要看赛道适配、作品完整性、创新性、准确性和方案 AI 原生性。 |
| 初赛晋级比例大概多少？ | 以官方后续通知为准。 |
| 复赛和决赛是什么形式？ | 通常会有导师指导、优化提交和路演答辩，具体以官方安排为准。 |

## 4. 奖励与资源

| 问题 | 回答 |
|---|---|
| 有奖金或其他奖励吗？ | 有，具体以赛程和官方奖励说明为准。 |
| 比赛会提供模型 API 额度吗？ | 不会统一提供，建议按规则自行准备。 |

## 5. 当前建议

1. 先保证 Demo 能顺利打开、能顺利讲完整主线。
2. 录屏时优先展示“分析 -> 视角切换 -> 时间轴 -> 无界能力”。
3. 说明文档里清楚写明哪些是已落地，哪些是后续计划。
